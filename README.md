# 🪿 Brisket Quest

A pixelated 3D goose game where you explore a charming countryside village, gather ingredients, and cook the perfect brisket.

## Play

Open `index.html` in your browser — no install or server needed.

## Controls

| Key | Action |
|-----|--------|
| `W A S D` | Move |
| `Mouse` | Look around |
| `E` | Interact / Pick up |
| `Space` | Honk 📣 |
| `Esc` | Release mouse |

Click **🪿 HONK TO START** to begin. The game captures your mouse pointer for look controls — press `Esc` to release it.

## Goal

**Cook a brisket.** To do that you need to:

1. **Explore** the village and surrounding nature to find all the ingredients
2. **Pick up** each ingredient by walking close and pressing `E`
3. **Find the recipe** hidden somewhere in the world — it tells you the correct order to apply ingredients
4. **Head to the Cooker Station** (the stone structure in the village)
5. **Follow the recipe** — interact with the station and apply ingredients in the right order, then cook

## Tips

- 🏡 You can enter every house — push open the doors and explore inside
- 🌸 Flowerbeds are off-limits (geese have manners)
- 🗺️ Check the minimap (top-right corner) to orient yourself
- 🧾 Your inventory is shown at the bottom of the screen
- 🐦 Listen to the birds — the world is bigger than it first appears

## Built With

- [Three.js](https://threejs.org/) — 3D rendering
- Web Audio API — ambient nature sounds & honking
- Vanilla HTML/CSS/JS — no build step
